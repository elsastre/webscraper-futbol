# AUF Analyzer

Backend: FastAPI + paquete webscraper_futbol
Frontend: React (Vite)

## Backend (API)

1. cd backend
2. . .venv\Scripts\Activate.ps1
3. uvicorn api:app --reload --port 8000

API:
- http://127.0.0.1:8000/docs
- GET /standings
- GET /standings/refresh

## Frontend

1. cd auf-frontend
2. npm install
3. npm run dev

Frontend:
- http://localhost:5173/
